[스크래치 중간고사 대비]

1주차
- 스크래치 = MIT 미디어랩에서 개발한 교육용 프로그래밍 언어
- 블록으로 형상화된 명령어들로 시작적인 방식으로 프로그래밍
- 교육목표 : 창의적으로 생각하기, 논리적으로 추론하기, 다른 사람들과 협동하기
- 스크립트란?
	- 영화나 연극에서 사용되는 대본,각본을 의미
	- 연기자들은 이 스크립트에 적혀 있는대로 행동
	- 컴퓨터 분애에서는 어떤 응용프로그램이 수행해야 할 규칙 또는 명령들을 적어 놓은 것
- 스크립트 언어 : 스프라이트와 무대제공 (스트라이트와 무대의 뜻 알기)
- 스크래치의 특징
	- 레고 블록을 조립하듯이 프로그래밍 가능
	- 여러 기능의 블록들을 이용한 그래픽 기반의 프로그래밍
- 스크래치 실행 첫화면 잘 알아두기(첫화면 설명 알아두기)
- 오프라인 에디터의 장점: 웹이 연결되어 있지 않아도 사용 가능
- 에디터 사용 기능 알아두기(메뉴, 인터페이스, 스크립트 작업창, 실행창, 스프라이트/무대관리창 등)
- 스크래치 블록의 분류 기준(동작, 형태, 소리, 이벤트, 제어, 감지, 연산, 변수, 내블록(함수))
- 확장기능 알아두기
- 스크래치 블록의 프로그래밍 요소별 분류 알아두기(문장, 자료/변수, 조건, 제어, 이벤트, 절차 블록)
- 쥐와 고양이 프로그램 실행하는 방법 알아두기
- 위쪽 0, 오른쪽 90, 아래 180, 왼쪽 -90
- 무대에서 사용할 수 있는 메뉴 알아두기(어떤것들을 사용할 수 있는지)
- 무대에서 변수를 만들면 어떤 변수가 되는지 알아두기
- 학습평가 알아두기
